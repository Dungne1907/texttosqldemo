# DataChat

Hướng dẫn sử dụng đầy đủ nằm trong **[README.html](./README.html)**.

- Mở trực tiếp file `README.html` trong trình duyệt, hoặc
- Sau khi chạy `node serve.js`, truy cập [http://localhost:5500/README.html](http://localhost:5500/README.html)

## Khởi động nhanh

```bash
npm install
# Tạo .env cạnh serve.js (DEEPSEEK_API_KEY hoặc GEMINI_API_KEY)
node serve.js
```

Ứng dụng: [http://localhost:5500](http://localhost:5500) · Chi tiết kỹ thuật: [REVIEW.md](./REVIEW.md)
